Inline Elements Default Behaviour

  • Inline elements don't start on a new line;
  • Inline elements only take up as much width as needed;
  • Width and height properties have no effect on inline-level elements
Inline-level elements Example
<a href="google.com">Google</a> Google Bing
<button>Click me!</button>
<input type="text">
<textarea name="" id="" cols="10" rows="5"></textarea>
<span>This is a span 1.</span> This is a span 1. This is a span 2
